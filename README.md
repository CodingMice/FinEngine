#Fine Engine

##简介
使用opengles进行视频图像(主要来源于摄像头)的处理和输出，摄像头采集数据-美颜、人脸识别-滤镜-录制-输出视频帧数据。

##工作流程
摄像头采集数据 data->{custom操作，比如美颜、人脸识别}->美颜、滤镜处理，输出纹理和data->渲染+录制+输出data到其他地方，比如U3D。
